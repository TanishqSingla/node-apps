# Usage
There are various commands listed in the help section in the app
### To view help section
`node app.js --help` 
### To add a note
`node app.js add --title="<title>" --body="<content>"`
### To remove a note
`node app.js remove --title="<title>"
### To list notes
`node app.js list`
### To read a note
`node app.js read --title="<title>"`
